Submission Guidelines

Before I start, let's get over the submission guidelines and the general principles in doing an assignment.
  1. First, my belief is that you learn best when you do things, yourself. You also learn when you do things in a group, but it becomes hard for me to give marks to figure out who has done what. So as a compromise, please work within the group, and not across a group.
  2. When you submit, please document individual percentages such as Student 1: 80%, Student 2:100%, Student 3:10%. In this example, the second student will get full marks (10/10) and the first student will receive 8/10.
  3. When you submit your assignment, you expect me to do a fair evaluation. This is usually difficult - For about 100 students, you might want to calculate the amount of time I have to spend, EVEN if I spend only 10% of the time you spent on it. For example, you might have spent about 180 minutes on this assignment (including reading this part :) So the best way of submitting an assignment is to follow the instructions VERY VERY carefully - the same way you probably filled out your JEE form. Any mistakes - of any kind - clerical, typographical, will attract penalties (i.e., reduced marks).
  4. You are going to create a web page that documents your work in this task. However, don't make it public. Submit it on moodle as a rollnumber.tgz (preferred) or .zip file (no .rar please, and no email please). Use the ASC format for your roll number (i.e., with leading zero if applicable). Be sure to note that moodle does not have infinite size, so prune your submission. DO NOT SEND JUNK FILES THAT ARE IRRELEVANT IN THE ASSIGNMENT (mac users especially to note). Any links within the web page should be to a local file - not absolute links. Also tgz the entire contents, but do not include any binary files. Do include makefiles, where appropriate, to make my task easy. Do include a readme.txt (telling me whatever you want to tell me), and the honor code.

The Tasks

  1. The purpose of this task is to able to present your work on a web page. You might have seen a gazillion web pages; wait, we are not going to make a super duper page. We will do just enough to get the basics of a web page. A modern web page has these principal components: CSS, Images, Javascript, and Forms. We won't do forms yet but we will do other stuff.

    2. CSS: A basic guideline is to separate content and form. The markup language tells us the browser what to write, and also how to mark them up, i.e., display. The Cascading Style Sheet (CSS) is the preferred way to decorate a page.
      Learn to style the html elements by using inline styling, included style blocks in the same html file and separate style file. Style one html element each a) inline, b) using a style block within in the html file and c) using a separate CSS style file. The style file must be called index.css. Use the same stylesheet for all webpages in this task.
      How we will score you: This section has 30 marks

      • 10 each for the three parts mentioned above.
    3. Javascript: A web page like the one you have so far is pretty static. We want to be able to interact with the web page. One way of doing this is using Javascript. This is a long topic but we want to do the real basics.
      (a) Create a new page called contactMe.html. Introduce the <script> tag to enable javascript, and create a "button" element. If you click the button, your coordinates should be revealed to the users. Feel free to place whatever little or large information here, we won't worry about the content. However, on clicking the button again, the contact information should disappear. Your code should be in a file called index.js and contain a function called toggle(). There might be a variety of ways of making this feature happen, we have used style.display == none to disable display unless clicked. See this picture.
      (b) What sets your HTML page apart from one that might be made in a typical HTML shop? Mathematics. Use the MathJax library to make beautiful math. I am especially interested in learning from you what you consider interesting mathematics formula. How does a casio calculator figure out sin−1 23? Write 3-4 formula that you love, be it physics, chemistry or mathematics or ...
      How we will score you: This section has 40 marks. Try to be creative in your math formula. If you use the same math formula as some other group, then something is wrong? Maybe?.
