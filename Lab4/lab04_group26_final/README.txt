Targets:

all: default target
setup: creating directories
check: checking whether Box2D installed
executable: making executable file cs251_exe_26
makobj: creating object files only
mstat: creating static library
mshare: creating shared library
exelib: creating executable cse251_exelib_26 using libraries
clean: clean files to return to pre-compiled state
distclean: clean files to return to original state

Instructions:
1. use "make -f makefile_group26" to check whether BOX2D installed or not.
2. use "make -f makefile_group26 makobj" to create object files only.
3. use "make -f makefile_group26 executable" to create executable cse251_exe_26.
4. use "make -f makefile_group26 exelib" to create executable cse251_exelib_26 using libraries.
5. use "make -f makefile_group26 clean" to clean and return to pre compiled state.
6. use "make -f makefile_group26 distclean" to cleand and return to original state.
