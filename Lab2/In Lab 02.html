In Lab 02

  • Please write (only if true) the honor code. If you used any source (person or thing) explicitly state it. You can find the honor code on the web page.

The Tasks

  1. Set up the files you submitted in the final submission of Lab01 so that they can be accessed using http://www.cse.iitb.ac.in/~yourname. In brief, you need to set up your public_html directory on mars.cse.iitb.ac.in for this to work. Comment on your experience in the readme.txt file.
  2. The goal here is to make sure you can make a simple LATEXdocument.
    • First learn how to make a table in LATEX. Create a three column table. The headings for the table has the group name, the roll number, and your name. We want to see the output of your name in Devanagari. (Write only the first name of group members, i.e., we want to see only one word here.)
    • (Extra credit - no marks) After having fun in writing your name(s), try to duplicate this image which has a famous, quintessentially Indian quote. We use the word "try" here to stress that it should be readable but need not be exactly the same.
    Gentle note: We recognize that all of you may not know Devanagari, but the teaching team does not know all the languages and would thus be unable to grade if the answer is not recognizable. If you need any help, please feel free to ask.
  3. Finally, you have to learn how to insert images in LATEX. So use the lab01 images, and insert them in such a way that all of them fit in one row, and the aspect ratio is pleasing. Do not put the images vertically in one column or two rows. An example is shown below.

What to Submit

You should submit the source hindi.dn and the final pdf output and the readme.txt file (containing the individual percentages and honor code). We will compile it and use images from the first lab. Don't resubmit the images, and other intermediate files, and the LATEXsource file hindi.tex

Scoring

  1. The public_html part carries 10 marks.
  2. Each of the other two parts carry 20 marks each.
